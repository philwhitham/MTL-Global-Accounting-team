## Entity Information

- **Company**: Shopify Financial Services Inc.
- **NMLS ID**: 2689562
- **Report Type**: MSB Call Report (Money Services Business)
- **Business Type**: Stored Value (Shop Pay Wallet Balance)

## Files in Each Quarter Folder

1. **`SHOPIFY_FS_[Quarter]_[Year]_MSB_REPORT.xml`** - The actual XML file for NMLS submission
2. **`[Quarter]_[Year]_REPORT_SUMMARY.md`** - Human-readable summary with financial data and checklist

## Submission Process

1. Generate XML from NetSuite data
2. Internal review (Finance, Legal, Compliance)
3. Validate XML against XSD schema
4. Submit to NMLS portal
5. Monitor for acceptance/rejection
